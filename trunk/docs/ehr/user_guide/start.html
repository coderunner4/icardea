How to start

This section describes the steps for starting up the components of the EHR interoperability framework, after they have been successfully installed. For the installation guide please check the How to install section.

First of all you have to launch the mongoDB database server. You have to launch a command prompt and cd to the folder where you have unzipped the distribution. For the now on I am assuming that you have unzip MongoDB in the c:\mongodb directory. In the home directory of the MongoDB you can find a bin subdirectory that contains various executable files

  • If you have not installed MongoDB as a Windows service, you can launch (in the command line, after cd to the bin directory of the mongoDB) the mongod executable by issuing:

    mongod -f mongodb.conf

    You can now check the web interface of MongoDB at http://localhost:28017/ to see its status.

  • Start XDS by issuing (in a command line window):

    python xds.py

  • Start Update Broker by issuing (in a command line window):

    python ub.py

  • Start the iCARDEA Patient Index in a command line window (after cd in the pixpdq directory) by launching maven as follows:

    maven run

  • Optionally you can also start the ATNA Audit Repository as described here, i.e. by issuing:

    java -jar openatna-1.2.1-SNAPSHOT.jar

    in the OpenATNA installation folder.

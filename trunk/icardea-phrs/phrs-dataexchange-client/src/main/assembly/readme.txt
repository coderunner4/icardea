This directory contains examples and validation schemes for the iCardea related 
models.

vital_sign_example.rdf - this is just an example, this is a vital sign 
serialized in XML/RDF.

vital_sign_scheme.rdf - the validation scheme for the vital sign, it provides 
a class and seven properties properties. 

observation_example.rdf - this is just an example, this is an observation 
serialized in XML/RDF. The Observation can be used for Problems, Risks, 
Activities of Daily living and Observation.

medication_example.rdf - this is just an example, this is a vital medication 
serialized in XML/RDF.
